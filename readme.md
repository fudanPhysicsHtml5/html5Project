## 功能
1. 下载对应html文件
2. 注册，登录，登出，修改密码
3. 管理员页面：1.管理上传项目：打开，发布项目，删除项目 2.管理已发布项目：打开，删除项目
4. 为管理员添加登录页面
5. 考虑添加删除的评论，同步到用户上传界面

## 实现：
1.使用django作后端
2.前端使用bootstrap框架
3.暂时使用django自带SQLite3数据库
